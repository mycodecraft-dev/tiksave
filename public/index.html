Tentang Kami

TikTok Downloader adalah alat online yang memungkinkan pengguna untuk mengunduh video TikTok tanpa watermark secara gratis. Platform ini dirancang untuk memberikan pengalaman download yang mudah, cepat, dan aman.

Kami berkomitmen untuk menyediakan layanan yang berkualitas tanpa biaya kepada semua pengguna. Platform ini dikembangkan dan dipelihara oleh tim developer yang berpengalaman.

Cara Kerja

  1. Pengguna memasukkan URL video TikTok
  2. Sistem mengirim permintaan ke API TikTok
  3. API memproses dan mengembalikan data video
  4. Sistem mengekstrak link download tanpa watermark
  5. Pengguna dapat mengunduh video dengan mudah

API yang Digunakan

Kami menggunakan API dari Endpoint Hub untuk mengambil data video TikTok:

https://endpoint-hub.up.railway.app/api/downloader/tiktok?prompt={URL_TIKTOK}

API ini menyediakan data lengkap video TikTok termasuk URL download tanpa watermark, cover image, dan audio terpisah.

Dokumentasi API

Panduan lengkap untuk menggunakan API TikTok Downloader

GET /api/download?url={URL_TIKTOK}

Endpoint untuk mengambil data video TikTok berdasarkan URL.

Parameter:

Parameter Tipe Wajib Deskripsi Response
url String Ya URL video TikTok yang valid (contoh: https://vt.tiktok.com/ZSfsDhoBk/) JSON{}

Contoh Request:

GET /api/download?url=https://vt.tiktok.com/ZSfsDhoBk/

Contoh Response:

{ "ok": true, "type": "video", "creator": "Farel Alfareza", "prompt": "https://vt.tiktok.com/ZSfsDhoBk/", "video": { "title": "arti arti secret kode #fyp ", "author": "5 DESEMBER GUE ULTAH", "region": "ID", "cover": "https://p16-sign-va.tiktokcdn.com/...", "no_watermark": "https://v16m.tiktokcdn.com/...", "no_watermark_hd": "https://v16m-default.tiktokcdn.com/...", "music": "https://v19-ies-music.tiktokcdn.com/..." } }
GET /api/statistik

Endpoint untuk mengambil data statistik penggunaan platform.

Contoh Response:

{ "ok": true, "totalDownloads": 1250, "dailyStats": { "labels": ["2023-12-05", "2023-12-06", "2023-12-07", ...], "data": [45, 67, 89, ...] }, "monthlyStats": { "labels": ["2023-07", "2023-08", "2023-09", ...], "data": [450, 520, 600, ...] } }

Contoh Penggunaan

JavaScript (Fetch API)

// Mengambil data video TikTok fetch('/api/download?url=https://vt.tiktok.com/ZSfsDhoBk/') .then(response => response.json()) .then(data => { if (data.ok) { console.log('Judul:', data.video.title); console.log('Download URL:', data.video.no_watermark); } });

Python (Requests)

import requests url = "https://localhost:8080/api/download" params = {"url": "https://vt.tiktok.com/ZSfsDhoBk/"} response = requests.get(url, params=params) data = response.json() if data.get("ok"): print(f"Judul: {data['video']['title']}") print(f"URL Download: {data['video']['no_watermark']}")
